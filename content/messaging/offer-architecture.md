# Offer Architecture

## Primary Offer: Efficiency Assessment

**Price:** $497 one-time
**Format:** 45-minute working session with Josh (may extend to 60)
**Audience:** Any contractor, any size, including early-stage
**Primary ICP:** Jim and Ethan

**What happens:**
1. Map the current lead flow (calls, forms, CRM, estimates, follow-up)
2. Find the leaks (missed calls, broken follow-up, no reviews, stale website)
3. Rank by impact and effort
4. Identify the first useful fix
5. Hand off a ranked action list with clear next steps

**What the client leaves with:**
- A ranked fix list: do now, do later, ignore for now
- Which fixes are free using tools they already have
- Which fixes Josh can help implement (optional)
- A plain-English next step for each key area

**Implementation after:**
- Some fixes are free (process changes or unused CRM settings)
- Missed-call text-back and basic automations: ~$100/month
- Blog content: ~$100/month
- AI phone coverage: setup fee + ~$200-300/month depending on volume
- Website or CRM work: quoted as a project
- Nothing automatic. Nothing bundled without agreement.

**Satisfaction commitment:**
If the client does not leave with at least one clear fix worth more than $497, Josh will make it right.
